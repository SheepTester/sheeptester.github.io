Convert rich text in clipboard into Markdown

Are you infatuated with your LLM and wish it could see the same fancy text formatting you see when you copy paste documentation into it? Or maybe you're working with Markdown for some other, more innocuous purpose.

This tool takes the formatted rich text as HTML that you paste from your clipboard and turns it into Markdown. Try it by selecting the entire page, copying, and pasting (ctrl/cmd + A, C, V).

I previously used Paste to Markdown, but I didn't like the Markdown formatting it produces. This tool uses Prettier to clean up the Markdown.

If you want to clean up colors and fonts from your clipboard before pasting into a different rich text editor like Google Docs or an email, you should use the font & color remover.
